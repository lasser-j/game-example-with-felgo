# Invasion – Simple Game Example with Felgo

This repository contains a small cross-platform shooter game called **Invasion**, created using [Felgo](https://felgo.com).

It is intended as a step-by-step example for learning how to build games with Felgo, including modular entities, animations, collision detection, and game logic.

## Demo & Documentation

The complete **HTML documentation** of the game tutorial is included in this repository.  
You can view it online via [GitHub Pages](https://lasser-j.github.io/game-example-with-felgo/html/how-to-create-a-simple-game-with-felgo.html).

Or locally, download **docs/html/** and open the HTML file **how-to-create-a-simple-game-with-felgo.html** in your browser.
